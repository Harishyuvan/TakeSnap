# TakeSnap - Chrome Screenshot Extension 📸

TakeSnap is a lightweight Chrome Extension that allows users to capture screenshots directly from any website and crop the captured area easily.

It is designed to provide a fast and simple way to capture specific parts of a webpage without needing external tools.

## 🚀 Features

- Capture screenshots from any website
- Crop selected areas of the screen
- Simple and minimal user interface
- Fast screenshot capture
- Lightweight Chrome extension

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- PowerShell
- Chrome Extension APIs

## ⚙️ Installation

1. Clone this repository

2. Open **Google Chrome**

3. Go to **chrome://extensions**

4. Enable **Developer Mode**

5. Click **Load Unpacked**

6. Select the **TakeSnap project folder**

The extension will now be installed in your browser.

## ▶️ How to Use

1. Open any website in Chrome
2. Click the **TakeSnap extension icon**
3. Capture the screenshot
4. Crop the desired area
5. Save the screenshot

## 🎯 Project Goal

The goal of this project is to create a simple Chrome extension that helps users quickly capture and crop screenshots while browsing websites.
